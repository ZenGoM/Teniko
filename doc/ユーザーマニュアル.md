# Teniko(TiffToPdfConverter) ユーザーマニュアル

## 1. 概要
`Teniko(TiffToPdfConverter)` は、レガシーな「Wangアノテーション（コダック・アノテーション）」が付与されたTIFFファイルを読み込み、PDFファイルへ変換するコマンドラインツールです。
TIFF画像上のWangアノテーションデータを抽出し、モダンなPDFの標準注釈（アノテーション）として埋め込みます。

## 2. 動作環境
本プログラムを実行するには、以下の環境が必要です。
- **OS**: Windows (コマンドプロンプトまたはPowerShell)
- **ランタイム**: .NET 8.0 SDK (開発・ビルド時) または .NET 8.0 Runtime (実行時)

## 3. 実行方法
コマンドプロンプトやPowerShell等のコマンドラインから、入力となるTIFFファイルのパスと、出力したいPDFファイルのパスを指定して実行します。

### 実行コマンドの形式
```powershell
.\teniko.exe <入力TIFFファイルのパス> <出力PDFファイルのパス>
```

### 実行例
ビルドされた実行ファイル (`teniko.exe`) があるディレクトリで以下のように実行します。
```powershell
.\bin\Debug\net8.0\teniko.exe C:\input\sample.tif C:\output\result.pdf
```
※開発環境から `dotnet` コマンドで直接実行する場合は以下のようになります：
```powershell
dotnet run -- sample.tif result.pdf
```

## 4. 主な機能と特徴

### アノテーションのPDF注釈化
元のTIFFファイルに含まれるWangアノテーションは、画像に直接焼き付け（フラット化）られるのではなく、**PDF標準の「注釈（Annotations）」として変換・付加**されます。
- テキストや付箋（Note）はPDFのテキスト注釈（コメントアイコン等）になります。
- 直線、ハイライト、矩形などの図形アノテーションも、それぞれPDFの図形注釈として出力されます。
これにより、出力されたPDFをAcrobat Reader等のPDFビュワーで開いた際、注釈パネルからアノテーションの表示/非表示の切り替えや、個別の削除を行うことが可能です。

### 変換状態のログ出力
TIFFファイルの仕様上、保存に使用したソフトによってはWangアノテーションデータがファイルから消失している場合があります。
本プログラムは実行時にコンソール画面へ以下のようなログを出力し、アノテーションデータの有無を可視化します。
- **データが存在し変換した場合:** `[Debug] Page X: WangAnnotationDataを検出しました (〇〇 バイト)。`
- **データが存在しない場合:** `[Debug] Page X: WangAnnotationData (タグ32932) は見つかりませんでした。`
もしPDFにアノテーションが反映されていない場合は、このログを確認することで、元ファイルにデータが存在していたかどうかを切り分けることができます。

## 5. 注意事項・既知の仕様
- **表示位置のズレ:** 元のTIFF画像が回転（Orientationタグ）して保存されている場合など、当時のソフトウェアの仕様上の問題で、変換後のPDF上でアノテーションの配置位置が元の見え方とズレるケースがあります。
- **テキストの文字化け:** Windows 95〜98時代の日本語環境（Shift-JIS）で作成されたテキストアノテーションを対象にデコードを行いますが、作成時の環境やデータによっては正しくテキストが変換できず文字化けする可能性があります。
